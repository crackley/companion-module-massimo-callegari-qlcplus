# companion-module-qlcplus

This module allows you to control QLC+ lighting software using Bitfocus Companion.

## Features

- Control QLC+ functions (start/stop)
- Control Virtual Console widgets (buttons, sliders, cue lists)
- Get channel values and widget status
- Simple Desk channel control

## Setup

1. Add the QLC+ module to your Companion setup
2. Configure the module with the IP address and port of your QLC+ instance (default: 127.0.0.1:9999)
3. Make sure QLC+ is running and the Web API is enabled

## API Information

This module uses the QLC+ WebSocket API. For more information, see the QLC+ documentation.
